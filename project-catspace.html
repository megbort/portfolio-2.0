Intro

Catspace is one of my recent creations. I've been learning Angular over the past several months and I wanted a project to demonstrate and experiment the skills I've learnt. The project idea is to create a social media type platform with a twist, it's for cats! The concept gives me room to add in a variety of custom and Angular Material UI components, plus cats are cute to look at.

Process

I first made some basic wireframes in Figma to help me plan out the layouts, design and colour palette. Since I knew I wanted this to be built in Angular, I didn't look around much for a framework or JS libraries. I decided to keep everything close to the Angular / Google technologies by using Angular Materials for some UI components and Google fonts / material icons. Also, I started using Tailwind CSS recently and it's now my go to CSS framework, so I would be adding this in the project as well. Once this was all planned out I started setting everything up, making the new app, adding a new git repository and adding dependencies.

At this moment I've added some components, pages and routing, but I plan to continue adding more and more to the project as time goes on. I want it to become bigger and better as my skills with Angular continue to grow.
